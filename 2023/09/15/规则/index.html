C#变量命名法 运算符 占位符 逻辑运算符

一、C#变量命名法
帕斯卡命名：当有多个单词组成变量或方法 每个单词的首字母大写
类名 属性 方法名、接口

Camel 驼峰命名 多个单词出现，字首字母大写，其他小写
Console.WriteLine(“”);

C#变量只能由数字、字母和下划线组成，但是不能用数字开头
尽量做到见名知意

二、运算符
1、算数运算符 + - * / %
2、关系运算符 < > == <= >= !=
3、逻辑运算符 && || ！与 或 非
4、赋值运算符 =

三、占位符的使用
1、用“{数字}”的形式表示变量
Console.Write(“账号{0}”，name);
Console.Write(“账号{0}，密码{1}”，name,password);
2、直接在“{}”里写变量名,前边加$
Console.Write($”账号{name},密码{pasword}”);
